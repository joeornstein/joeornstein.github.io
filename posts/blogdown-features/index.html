Blogdown Features

  • May 10, 2020
  • Categories: R
  • One minute read

Any website built with Hugo and blogdown will inevitably feature one blog post about all the cool features of Hugo and blogdown. This is mine.

Observe, if you will, that we can embed a shiny dashboard into a static website with RMarkdown. Thanks to Yang Liu!

knitr::include_app("https://vac-lshtm.shinyapps.io/ncov_tracker/?_ga=2.126674445.1246084674.1588877732-1324960155.1586370896", height = "600px")

The same knitr function also works for slides:

knitr::include_url('https://timmastny.rbind.io/slides/first_presentation#2')
